# Wei & Mei · Wedding RSVP

A modern Chinese New Year–themed wedding RSVP site built with React, TypeScript, and Vite.

## Features

- Click-to-open envelope landing animation
- Full invitation page with celebration details
- Prenup photo gallery (Unsplash stock photos)
- Chinese motif decorations (lanterns, clouds, blossoms, 囍)
- RSVP form (name, email, attendance, message)

## Getting started

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```
